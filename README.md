Real-time risk intelligence platform using ensemble ML with sub-150ms inference and probability calibration.

Interactive system for sentiment analysis, text generation, and sequence modeling using RNN/LSTM.

### ❤️ ECG Anomaly Detection

Unsupervised anomaly detection using autoencoder architecture with ROC-AUC 0.817.

### 👁️ Vision Anthropometrics

Modular full-stack event platform with backend architecture and controlled AI-driven theme generation.
